### Docker usage

    Build server
        docker-compose -p nodejs-ts-abcdin build

    Start server
        docker-compose up -d

    Stop server
        docker-compose down

### Standalone usage

    npm run dev

### Testing

    To run unit testing
        npm run test

    To run unit testing coverage
        npm run test_cov

### Environment variables

To modify/add configuration via environment variables, use the `.env` file, which contains basic app configuration.
